# 飞书多维表格功能调研

> 调研日期：2026-03-12

## 产品定位

飞书多维表格（Lark Base）不只是一个"表格"，而是一个 **AI 驱动的低代码业务系统搭建平台**。核心思路：**数据库 + 多视图 + 自动化 + AI + 低代码应用**。竞品对标 Airtable / Notion Database，但 AI 集成深度和工作流能力更强。

## 功能全景

### 1. 数据管理基础

- **20+ 字段类型**：文本、单选/多选、日期、附件、公式、关联、进度、货币、评分、电话、邮箱、地理位置、条码、按钮、流程、自动编号等
- 单表最多 **200 万行**，支持自助扩容
- **1,000 人**同时编辑
- **100 万行归档表**（存储历史数据）
- 一键导入 Excel

### 2. 多视图展示

同一份数据可以用不同方式呈现：

| 视图 | 用途 |
|------|------|
| 表格视图 | 默认，结构化数据浏览 |
| 日历视图 | 按日期排列事件 |
| 甘特视图 | 项目时间线管理 |
| 表单视图 | 数据收集，支持预填默认值 |
| 看板视图 | 任务状态流转 |

- 支持独立分享单个视图
- 视图保护（防止误操作）

### 3. 仪表盘（Dashboard）

50+ 图表组件，对标专业 BI 工具：

- **基础图表**：柱状图、折线图、饼图、面积图、组合图
- **高级图表**：漏斗图、雷达图、词云图、NPS 图
- **数据组件**：指标卡、排行榜、进度图、倒计时
- **分析功能**：数据透视表、切片器、图表联动、TopN 筛选、行列切换
- 多种专业主题和自定义背景
- AI 创建仪表盘

### 4. 工作流 & 自动化

从简单自动化到复杂业务流程：

- **触发条件**：记录变更、定时、接收飞书消息、点击按钮
- **多分支节点**：2~10 个分支，类似 IFS 函数
- **流程公式**：在流程中对数据/变量进行计算
- **AI 节点**：自动生成文本、图片等
- **AI 节点捷径**：封装高频场景，简单配置即可使用
- 发送消息卡片（含可交互按钮：跳转、新增/修改记录）
- 按钮支持全局点击次数限制（如抢单场景）

### 5. AI 能力

AI 全面集成到各个环节：

| 功能 | 说明 |
|------|------|
| AI 字段捷径 | 翻译、分类、总结、信息提取、内容生成 |
| 浮窗工具栏 | 点击单元格一键调用 AI（填充、润色、翻译、标签提取） |
| AI 侧边栏 | 对话式交互，自然语言操作表格 |
| AI 推荐关联 | 智能匹配关联记录 |
| AI 创建仪表盘 | 对话式生成数据看板 |
| AI 搭建工作流 | 对话式配置自动化流程 |
| 三方 AI 捷径 | 接入第三方 AI（批量生成营销图、法律文件提取等） |

### 6. 应用模式（低代码搭建）

- 拖拽式构建业务系统，无需代码
- 一个应用可关联多个多维表格
- 自定义界面和交互流程
- 独立的权限管理体系
- 支持预览、分享（链接/二维码）

### 7. 权限管理

- **高级权限**：角色化权限分配
- **行列级权限**：不同角色看到不同的行和列
- 成员间数据隔离
- 权限预览功能

### 8. 开放集成

- **80+ 连接器**：同步第三方系统数据到多维表格
- **多维表格空间**：集中管理多个多维表格文件
- **沙箱环境**：安全测试配置
- **记录详情页自定义**
- **API 开放接口**（REST API，支持 MCP Server 接入）

### 9. 公式 & 函数

- 完整的公式引擎
- 支持引用关联字段、地理位置等字段属性
- 统计函数与 CurrentValue
- 公式计算优化指南
